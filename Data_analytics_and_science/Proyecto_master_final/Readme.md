Proyecto final de Master: Visión Artificial
Contexto
Pontia Logista quiere automatizar sus procesos en el identificado del tipo de fruta además de mejorar su sistema para almacenar y gestionar sus datos
Expectativas del proyecto
El objetivo es proporcionar a Pontia Logista un sistema de almacenaje de datos adecuado, revisar y corregir errores en sus datos y automatizar la categorización del tipo de fruta
Los entregables incluyen:
- Jupyper Notebook que contiene un análisis detallado de los datos y la transformación para una BBDD SQL
- El esquema relacional y un archivo SQL con respuestas a preguntas del negocio
- Jupyter Notebook con el entrenamiento de un modelo de deep learning para la categorización del tipo de fruta
- Dashbord con KPIs utilizando Power BI
